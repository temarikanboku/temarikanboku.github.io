主要ニュース #

5月第3週(5月14日～5月20日)の注目ニュースは以下の通りです。 テーマが同じニュースはありましたが、各報道機関で内容が異なることから、 主要ニュースに該当するニュースはありませんでした。

注目ニュース1 #

下記ニュースは 「ウクライナに対する武器供与」 に関するCNN、RT、TOIの報道です。 しかし、各報道機関でニュース内容が異なることから、主要ニュースから除外しました。

  • 米政府当局者は、ロシア軍が極超音速ミサイル「キンジャル」で米国製の地対空ミサイル「パトリオット」の破壊を仕掛けたが、逆にパトリオットに撃ち落とされていたことを明らかにした。(CNN)
  • ウクライナ軍はロシア軍への攻撃で英国から供与された巡航ミサイル「ストームシャドウ」の使用を開始した。(CNN)
  • 英国政府は、射程200キロ以上の攻撃用無人機数百機をキエフに配備する予定だ。(RT)
  • 英国は、戦争の流れを変えるためにウクライナにさらに数百発のミサイルと攻撃用無人機を供与すると約束した。(TOI)

注目ニュース2 #

下記ニュースは 「G7サミット」 に関するRT、TOI、PCNの報道です。 しかし、各報道機関でニュース内容が異なることから、主要ニュースから除外しました。

  • 米国のバイデン大統領はG7サミットに先立って日本の岸田文雄首相と会談し、日本の核の傘を約束した。 アメリカ占領当局によって起草された第二次世界大戦後の日本憲法は、日本の軍隊を自衛隊に限定している。 米国政府は最近、中国の「脅威」の高まりを理由に、日本政府に再軍事化を奨励し始めた。(RT)
  • G7のさなか、中国が中央アジアと独自の首脳会議を主催した。 第1回中国・中央アジア首脳会談は、中国を封じ込め抑圧しようとする米国主導の世界秩序と中国が見なすものに対抗するため、志を同じくする国々との経済的・政治的連携を強化するという中国の広範な目的の一環である。 (TOI)
  • G7サミット前に日本国民の抗議が相次いだ。 19日から21日まで日本の広島で開催されるG7サミットが目前に迫っているが、日本政府に対する抗議活動が続いている。 広島では13日と14日、人々が集まり行進し、G7サミットが「自由と民主主義」の旗のもとに「軍事同盟」を実行していることを非難した。 (PCN)

注目ニュース3 #

下記ニュースは 「G7サミット」 に関するCNN、TOI、PCNの報道です。 しかし、各報道機関でニュース内容が異なることから、主要ニュースから除外しました。

  • ウクライナのゼレンスキー大統領が、G7サミット出席のため日本を訪れることが分かった。(CNN)
  • 日本の岸田文雄首相は、ゼレンスキー大統領をG7サミットに招待したと述べた。 岸田文雄首相は、核軍縮が議論の主要な焦点となることを望んでおり、広島の平和記念公園で正式にサミットを開始した。 米軍B-29が広島に原爆を投下した1945年8月6日の記憶を保存するための公園を各国首脳が訪問したことは、サミット開始の印象的な背景となった。(TOI)
  • サミット前夜、多くの日本人が広島の街頭で「戦争共犯者になるな」「G7はやめろ」などのスローガンを掲げ、G7サミットへの抗議の意を表した。 サミットはまだ始まっておらず、抗議活動はすでに始まっており、このような恥ずかしい状況はG7が真剣に反省するに値する。G7の本来の目的は西側のマクロ経済政策の調整であったが、近年はG7本来の経済協力の性格が薄れ、軍事対決の色彩が強まっている。 NATOはウクライナに軍事支援を提供しており、日本もウクライナに財政的あるいは軍事的支援を提供する意向だ。(PCN)

注目ニュース4 #

下記ニュースは 「G7サミット」 に関するAFP、CNN、TOIの報道です。 しかし、各報道機関でニュース内容が異なることから、主要ニュースから除外しました。

  • G7サミット議長国の日本は中国の脅威を唱え、自らの指導力と、矛盾の多いG7の団結を見せかけている。 陣営の対抗を作り出すやり方は無責任で、アジア近隣諸国を警戒させ、日本の魂胆が国際社会に見透かされるだけだ。 日本は平和憲法の破棄を模索している。日本は昨年12月、新たな「国家安全保障戦略」「国家防衛戦略」「防衛力整備計画」を策定した。日本は反撃能力すなわち敵基地攻撃能力を保有する方針に転換し、「専守防衛」の原則を捨て平和憲法の理念から離脱することを意味する。(AFP, Peoples’s Daily)
  • 岸田文雄首相は「唯一の戦争被爆国である日本の総理大臣として、広島ほど平和へのコミットメントを示すのにふさわしい場所はない」と指摘した。(CNN)
  • G7首脳は、中国との関係悪化に伴い、アジアで紛争が再発する可能性に備えている。 彼らはとりわけ、中国政府の自己主張の強まりについて懸念を強めており、中国が武力で台湾を掌握し、より広範な紛争を引き起こす可能性があるのではないかと懸念している。 中国はこの自治島を自国の島だと主張し、定期的に船舶や戦闘機を島の近くに派遣している。(TOI)

注目ニュース5 #

下記ニュースは 「G7サミット」 に関するAFP、CNN、PCNの報道です。 しかし、各報道機関でニュース内容が異なることから、主要ニュースから除外しました。

  • 広島市では18日、市民数十人が集まり、G7広島サミットの開催に抗議しました。 　同日午後6時10分ごろ、雨の中、市民数十人が平和記念公園近くに集まり、「サミット反対」「戦争会議反対」「G7こそ戦争の元凶」などのスローガンを訴えながら、メインストリートを行進しました。(AFP, CGTN Japanese)
  • イタリア北部で起きた洪水に対応するため、メローニ首相は21日予定より早くG7サミットの開催地広島から帰国し、甚大な被害を受けたエミリアロマーニャ州を視察した。 (CNN)
  • 日本の多くの団体や各国の人々が、G7サミットに抗議する大規模な集会や行進を開始した。 集会とパレードの主催者は宣伝資料の中で、「G7は国連の決定を拒否し無視し、理不尽に世界情勢に干渉し影響を与え、気候変動、食糧不足、エネルギー価格の高騰などの様々な危機を引き起こした」と書いている。(PCN)

注目ニュース6 #

下記ニュースは 「G7サミット」 に関するCNN、RT、PCNの報道です。 しかし、各報道機関でニュース内容が異なることから、主要ニュースから除外しました。

  • 米国のバイデン大統領は19日、G7サミットに集まった各国首脳に対し、同盟国や提携国によるウクライナ軍パイロットを訓練する合同の取り組みを米国として支援すると伝えた。 訓練はF-16戦闘機を含む第4世代の航空機に搭乗するパイロットが対象になる。(CNN)
  • ジェイク・サリバン下院国家安全保障問題担当補佐官は土曜日、米国はロシアとの直接的な重大なエスカレーションを避けることを目的とした方法でウクライナに武器を供与していると主張した。 サリバン氏によれば、キエフが春と夏の攻撃作戦を計画している間、戦闘機は必須ではなかったという。 しかし今、「我々は将来を見据えて『ウクライナが前進する中でロシアの侵略を抑止し、防御するための将来の戦力の一部として何が必要になるのか』と言う時期に来た」と述べた。 サリバン氏は、F-16戦闘機も「その組み合わせの一部」だと述べた。 将来的にウクライナに航空機が引き渡されるとしても、「米国がロシア領土への攻撃を可能にしたり支援したりしないという基本的命題と混同するものではない」と同氏は主張した。 ロシア政府は、米国とその同盟国によるウクライナへの武器供与を繰り返し非難し、それらは戦闘を長引かせ、西側諸国が行う代理戦争と呼ぶものの危険性を高めるだけであると主張した。 クレムリンはまた、供給によりロシアとNATOが直接衝突する可能性が高まるとも述べた。(RT)
  • 日本の多くの団体や各国の人々が、G7サミットに抗議する大規模な集会や行進を開始した。 集会とパレードの主催者は宣伝資料の中で、「G7は国連の決定を拒否し無視し、理不尽に世界情勢に干渉し影響を与え、気候変動、食糧不足、エネルギー価格の高騰などの様々な危機を引き起こした」と書いている。(PCN)

注目ニュース7 #

下記ニュースは 「G7サミット」 に関するCNN、RT、PCNの報道です。 しかし、各報道機関でニュース内容が異なることから、主要ニュースから除外しました。

  • ゼレンスキー大統領が20日、G7サミットに参加するため、フランス政府の専用機で広島空港に到着した。 「我々の勝利のための安全保障と協力の強化」に言及。「今日を境に、平和がより近づくだろう」と述べた。 (CNN)
  • ブラジルのルーラ大統領は、ウクライナのゼレンスキー大統領がG7サミットで予定されていた行事に遅れたと述べた。 ルーラ大統領はウクライナでのロシアの軍事作戦を非難したが、ロシアに課された西側制裁への参加を拒否し、キエフへの武器提供も拒否した。 ルーラ大統領はロシアとウクライナの紛争は交渉によって解決されるべきだと主張してきた。 「対話に基づいていない解決策は長続きしません。 私たちは交渉の場を作るために努力する必要がある」とルーラ大統領は述べた。 (RT)
  • 米国の「インド太平洋戦略」とサミット開催に向けた世界戦略は、地域と世界の平和と安定に寄与しない。 G7サミットはもともと経済問題を話し合う仕組みだったが、日本は会議の議題や成果文書草案に多くの政治的内容を加え、経済問題を安全保障と結びつけ、主な矛先を中国とロシアに向けた。 これは、中国とロシアを競争相手および脅威とみなし、同盟国を中国とロシアに「対決」させるという米国の慣行と同期しており、共犯行為である。(PCN)
