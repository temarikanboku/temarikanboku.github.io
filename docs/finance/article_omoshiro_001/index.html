財政赤字の安定化条件「ドーマー条件」の再考察

現在日本は大規模な財政出動によって大量の国債発行を余儀なくされています。 財務省によると 2022 年度末には普通国債残高が 1,029 兆円に上がると見込まれています。

国債は借金なのでいずれは返済をしないといけません。 1,000 兆円というと途方もない金額ですから、返済するとなると大変なことです。 国債発行をこのまま続けても日本の財政は大丈夫なのでしょうか。

この疑問に対して本論文は財政赤字の安定化の観点で考察を行っています。 本論文が提唱する財政の安定化条件によれば日本は財政破綻しない状況にあると述べています。 日本の場合、国債発行を続けても国債残高は安定化し、財政は安定化するというわけです。

これはとても心強く明るい結果です。 日本の国債の債務残高の心配は無くなりました。

・・・となればよかったのですが、本論文ではマネーサプライの増加を当てにしているので、 マネーサプライによっては財政が安定化するとは言い切れません。

結論として、本論文は日本が財政破綻しない根拠になるものではありませんでした。 財政の安定化条件について更なる研究が必要になることでしょう。

本論文は財政の安定化について興味深い要素を含んでいますので、 以下で本論文の主要なポイントについて述べていきたいと思います。

国債供給と国債需要 #

本論文は、国債市場の安定化(=財政の安定化)を議論する場合に、 政府の予算制約の式(=国債の供給式)だけでなく国債需要の式を考える必要があると述べています。

[予算制約の式(=国債供給の式)]

$$ \Delta B^S_t = (G_t - T_t) + r^B_t \times B_{t-1} - \Delta M_t \tag{1} $$

[国債需要の式]

$$ \Delta B^D_t = (b_0 + f_0) + f_1 \left[ r^B_t - \left\{ f^f_t + \frac{e^e_t - e_t}{e_t} \right\} \right] + b_1 (r^B_t - r^I_t) \tag{2} $$

変数は以下の通りです。

変数説明
$G_t$歳出
$r^B_t$国債金利
$B_t$国債残高
$\Delta B^S_t$新規国債の発行額
$T_t$税収
$\Delta M_t$マネーサプライ
$b_0$国内投資家による日本国債需要
$b_1$国内投資家による日本国債需要の利子係数
$f_0$外国人投資家による日本国債需要
$f_1$外国人投資家による日本国債需要の利子係数

均衡利子率 #

式(1)と式(2)から均衡利子率($r^B$)は次のように求められます。

[均衡利子率の式]

$$ r_t^{B*} = \frac{ (G_t - T_t) - \Delta M_t - (b_0 + f_0) + f_1 \left( r^f_t + \frac{e^e_t - e_t}{e_t} \right) + b_1 r^I_t }{ ( b_1 + f_1 ) - B_{t-1} } \tag{3} $$

本論文は均衡利子率の式の性質を以下のように説明しています。

#説明
1政府支出の増加($G_t-T_t$の増加)は国債発行を増大させ国債利子率の上昇を導く
2日本銀行による公開市場操作を通じたマネーサプライの増加は国債利子率を低下させる
3銀行、保険、年金等による国債需要の増加($b_0$の増加)は国債利子率を低下させる
4国債残高の増加($B_{t-1}$)は国債利払い費の増加となり、国債利子率は上昇する
5外国人が日本国債を売却した場合($f_0$が減少した場合)、国債利子率は上昇する

上記の説明は現実に起こりうる国債利子率の変化を説明していることから、 均衡利子率の式は確からしいものと考えられます。

財政安定化条件 #

本論文では財政安定化の条件を以下のように定めています。

  • 国債の増発により、新たな国債発行が徐々に減少する(安定化)
  • 国債の増発により、新たな国債発行が徐々に増加する(不安定化)

すなわち財政安定化は、$\frac{\partial \Delta B_t}{\partial B_{t-1}}$の値が正か負かで判定することができます。

財政安定化の式は式(1)と式(3)を用いて導かれます。

[財政安定化の式]

$$ \frac{\partial \Delta B_t}{\partial B_{t-1}} = \left( \frac{1}{1 - \frac{B_{t-1}}{b_1 + f_1}} \right) r_t^{B*} \tag{4} $$

財政安定化の式から、本論文では財政安定化の条件が以下になると説明しています。

  • $1/B_{t-1} < 1/(b_1 + f_1)$ならば、つまり$1 < \frac{B_{t-1}}{b_1 + f_1}$ならば財政は安定化する
  • $1/B_{t-1} > 1/(b_1 + f_1)$ならば、つまり$1 > \frac{B_{t-1}}{b_1 + f_1}$ならば財政は不安定化する

ここで気を付けないといけない所は国債金利の$r_t^{B*}$です。 上記の財政安定化の条件判定では$r_t^{B*}>0$を前提としています。 このことは本論文で特に説明がありません。 ところが、日本ではマイナス金利ですので、財政安定化の条件が真逆になってしまいます。

間違えないように以下に財政安定化の条件をまとめておきます。

条件条件式$r_t^{B*}>0$の場合$r_t^{B*}<0$の場合
1$1/B_{t-1} < 1/(b_1 + f_1)$財政は安定化する財政は不安定化する
2$1/B_{t-1} > 1/(b_1 + f_1)$財政は不安定化する財政は安定化する

日本のケース #

本論文では日本のケースを以下にのように述べています。

  1. $(b_1 + f_1) < B_{t-1}$となっている (条件 1 を満たしている)
  2. 財政の安定化の条件を満たしている
  3. 金利をマイナスにしている

ここで気を付けたいのは 3.のマイナス金利です。 マイナス金利の場合に条件 1 を満たすと、財政は不安定化してしまいます。 つまり、2.と 3.が矛盾することになります。

これは何らかの記載ミスのように見えます。 現時点において本論文は日本が財政破綻しない根拠になるものではありません。

シミュレーション #

財政安定化の式の定性的な性質のみを見てきましたが、 これだけでは実際に国債残高や新規国債の発行額がどうなるのか分かりません。 そこでシミュレーションによって各変数の値の推移を見てみます。 (小さいプログラムですが、githubで公開予定です。)

マネーサプライが一定で、金利が変動する場合 #

マネーサプライが多い場合 #

シミュレーションで使用したパラメータを以下に示します。 値はオーダが合えばよいというレベルで決めています。 簡単のためいくつかの変数を固定しています。

変数説明
$G_0$1000 兆円 ($G_t$の初期値)
$G_t-T_t$30 兆円
$\Delta M_t$35 兆円
$-(b_0 + f_0) + f_1 (r^f_t + (e^e_t-e_t)/e_t) +b_1r^I_t$0
$b_1 + f_1$500 兆円

シミュレーション結果を以下に示します。 マネーサプライ($\Delta M_t$)が多いとプラス金利になりました。 $\frac{\partial \Delta B_t}{\partial B_{t-1}}$の値が 0 以下なので財政の安定化条件を満たしています。 安定はしていますが国債残高は増える一方という結果です。 マネーサプライの増加を当てにしているので財政が安定化しているとは言い切れません。

(横軸の単位は年数です)

マネーサプライが少ない場合 #

シミュレーションで使用したパラメータを以下に示します。

変数説明
$G_0$1000 兆円 ($G_t$の初期値)
$G_t-T_t$30 兆円
$\Delta M_t$25 兆円
$-(b_0 + f_0) + f_1 (r^f_t + (e^e_t-e_t)/e_t) +b_1r^I_t$0
$b_1 + f_1$500 兆円

シミュレーション結果を以下に示します。 マネーサプライ($\Delta M_t$)が少ないとマイナス金利になりました。 $\frac{\partial \Delta B_t}{\partial B_{t-1}}$の値が 0 以上なので財政の安定化条件を満たしていません。 安定はしていませんが国債残高は減少しています(減少幅が拡大するという意味で安定はしていません)。 マネーサプライを減らせばマイナス金利に誘導でき、しかも国債残高は減少するという驚きの結果となりました。 これが本当ならば国債の債務残高の心配は無くなるのですが、こんなに簡単に行くのか疑問です。 なにかを間違えているように思えてしまいます。

金利が一定で、マネーサプライが変動する場合 #

$f_1$が小さい場合 #

シミュレーションで使用したパラメータを以下に示します。 $f_1$は外国人投資家による日本国債需要の利子係数です。

変数説明
$G_0$1000 兆円 ($G_t$の初期値)
$G_t-T_t$30 兆円
$r_t$0.5%
$-(b_0 + f_0) + f_1 (r^f_t + (e^e_t-e_t)/e_t) +b_1r^I_t$0
$b_1 + f_1$500 兆円
$f_1$100 兆円

シミュレーション結果を以下に示します。 $f_1$が小さいと$\frac{\partial \Delta B_t}{\partial B_{t-1}}$の値が 0 以上になりました。 財政の安定化条件を満たしていません。 安定せず国債残高は増える一方という結果です。

$f_1$が大きい場合 #

シミュレーションで使用したパラメータを以下に示します。

変数説明
$G_0$1000 兆円 ($G_t$の初期値)
$G_t-T_t$30 兆円
$r_t$0.5%
$-(b_0 + f_0) + f_1 (r^f_t + (e^e_t-e_t)/e_t) +b_1r^I_t$0
$b_1 + f_1$500 兆円
$f_1$200 兆円

シミュレーション結果を以下に示します。 $\frac{\partial \Delta B_t}{\partial B_{t-1}}$の値が 0 以下なので財政の安定化条件を満たしています。 国債残高は減少しています。 しかし、マネーサプライの増加を当てにしているので財政が安定化しているとは言い切れません。

まとめ #

日本が財政破綻しない根拠になるのではと思い、論文を紹介させていただきました。 本論文は日本のケースで財政の安定化条件を満たすと述べていますが、 現状のマイナス金利の場合は財政の安定化条件を満たさないことが分かりました。

そこで、なんとか財政の安定化条件を日本のケースに当てはめられないかと考え、 本論文の方程式をシミュレーションで分析してみました。 その結果、日本のケースにおいては、財政安定化条件を満たす場合にマネーサプライの増加が前提となるため、 マネーサプライによっては財政が安定化するとは言い切れないことが分かりました。

残念ではありますが、現時点において本論文は日本が財政破綻しない根拠になるものではありませんでした。 今後は財政の安定化の条件に加えて、マネーサプライの条件を考慮した財政の分析と研究が進められることを期待したいと思います。
